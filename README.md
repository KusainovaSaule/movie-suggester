# Movie Suggester

Приложение для выбора фильма на вечер. Нажимаешь кнопку — получаешь случайный фильм из списка с постером.

## Функциональность

- Рандомный выбор из 23 фильмов
- Адаптивный дизайн (мобильная и десктопная версии)
- Постеры к каждому фильму

## Технологии

- React + TypeScript
- Vite
- ESLint, Prettier
- GitHub Pages + Actions

## Деплой

Сайт доступен по адресу: https://KusainovaSaule.github.io/movie-suggester/

## Запуск локально

```bash
npm install
npm run dev
```
